# Mystore - E-Commerce Application  

Mystore is a modern e-commerce web application built with **Spring Boot 3**, **Thymeleaf**, and **MySQL**.  
It includes secure authentication, password reset via email, Razorpay payment integration, and is containerized with Docker for deployment on **Render**.

- Dockerized for cloud deployment
